# Analysis System
App URL : https://liatdavid2.github.io/angular_analysis_system:\
A dashboard app where you can upload a CSV file and view the data in graphs visualization.

Technologies:\
Front-end : Angular, Scss ,HTML5, Material-ui .\
Back-end : C# webapi
